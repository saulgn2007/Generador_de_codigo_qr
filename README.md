# Generador_de_codigo_qr

## 🔍Objetivos🔎 

El objetivo es desarrollar una aplicacion para la generacion y personificacion de codigos qr a traves de un enlace.

## 🎯Requisitos

- El codigo Qr debe poder ser personalizable
    

## 💻Tecnologias

La tecnologia usada es la siguiente:
- Para el desarrollo inicial y la elaboracion del proyecto hemos utilizado antigravity con su propio editor de codigo.

## 🖥️¿Como ejecutarlo?
Para ejecutarlo, solo deberemos ejecutar el index.html y empezara la ejecucion del programa.
